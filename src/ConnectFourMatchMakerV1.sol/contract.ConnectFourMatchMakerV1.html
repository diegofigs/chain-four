Structs

Request

Used to represent player match requests

Data layer for securing match deposits from players

struct Request {
    address player1;
    address player2;
    address token;
    uint128 amount;
    RequestStatus status;
}

GameMatch

Used to represent a game match, an extension enabling wagers for the base game struct

struct GameMatch {
    address token;
    uint128 amount;
    uint32 lastInteracted;
    bool claimed;
}
